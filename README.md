# 📚 Моя личная библиотека

Простое веб-приложение для управления списком книг.

---

[**🌐 Перейти на сайт Библиотеки**](https://john1rich.github.io/Library/)

### ✨ Функции:

- Добавление новых книг в список.
- Поиск книг по названию или автору.
- Сохранение данных в `LocalStorage` (данные не пропадают после перезагрузки).

### 🛠 Технологии:

- HTML5 / CSS3
- JavaScript (ES6+)
